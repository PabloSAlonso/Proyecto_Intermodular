# Project Context

Nombre: Muro de Usuarios

Descripción:
Aplicación social básica donde los usuarios pueden:

- registrarse
- iniciar sesión
- publicar mensajes
- ver publicaciones de otros usuarios

Clientes:

1. Web App (PHP)
2. Mobile App (Java Android)
3. Desktop App (C#)

Todos consumen la API central.

Flujo:

Usuario -> Cliente -> API -> Base de datos
