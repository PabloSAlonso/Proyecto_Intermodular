# API Specification

Base URL:

/Api/

Endpoints:

## Usuarios

POST /Api/usuarios/create
Crear usuario

POST /Api/usuarios/login
Login usuario

GET /Api/usuarios/get?id=1
Obtener usuario


## Publicaciones

POST /Api/publicaciones/create
Crear publicación

GET /Api/publicaciones/list
Listar publicaciones

DELETE /Api/publicaciones/delete?id=1
Eliminar publicación
